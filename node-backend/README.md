# Node.js Backend

This service satisfies the Review 3 Node.js backend rubric. It exposes CRUD APIs for activity log records and a read API for the audit entries generated by those CRUD actions.

## Run

```powershell
cd C:\Users\HP\Documents\Playground\source_project\node-backend
node server.js
```

The service runs on:

```text
http://127.0.0.1:7000
```

## APIs

- `GET /health`
- `GET /api/node/logs`
- `GET /api/node/logs/:id`
- `POST /api/node/logs`
- `PUT /api/node/logs/:id`
- `DELETE /api/node/logs/:id`
- `GET /api/node/audit-trails`

The FastAPI gateway exposes these APIs through:

```text
http://127.0.0.1:8000/node-api
```
